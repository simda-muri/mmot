## Multimarginal Optimal Transport (MMOT)

Code to accompany paper 

> B. Zhou and M. Parno (2022) "Efficient and Exact Multimarginal Optimal Transport with Pairwise Costs"

## Installation 

```bash 
pip install mmot
```

Or with a local copy of the source:
```bash
cd mmot
pip install -e .
```
