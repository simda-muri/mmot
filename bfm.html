BFM Wrapper API#

mmot.bfm_utilities.ascent_direction(rho, nu, kernel)#

Computes the ascent direction

g = Δ⁻¹(ρ − ν)

and the squared norm

|g|^2

This function does not change the values of rho and nu

mmot.bfm_utilities.c_transform(bf, dualVar, x, y, weight=1.0)#

Returns the c transform of the Kantorovich dual variable.

First, the dual variable is converted to convex form and then the bf module’s ctransform function is employed.

mmot.bfm_utilities.convex_conversion(g, x, y)#

Given the dual variable g(x) evaluated at the points x, this function returns the ϕ(x) = ½|x|² - g(x)
